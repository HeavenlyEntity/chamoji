# Welcome to Chamoji

> This is a modern alternative to gitmoji-changelog 

## Features
1. Configuration for all emojis and not just some

## Install

```bash
$ npm install --global chamoji
```


## CLI

```
$ chamoji --help

  Usage
    $ chamoji

  Options
    --name  Your name

  Examples
    $ chamoji --name=Jane
    Hello, Jane
```
